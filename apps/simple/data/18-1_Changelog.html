Notable Change in 18.1 (November 13, 2018)

  • Site Admin section for managing Categories (Trove) has been fixed.
  • SVN hooks have been implemented.
  • Kanban now properly shows tickets assigned to "nobody" when you group by assignee.
  • Sprints and Releases - Burndown charts rendering properly now.
  • Wiki - Added right-click context menu for following or copying a wiki link.
  • Installer now prompts user for license key information and asks if they'd like to start the docker containers.
  • Repository code indexing can be enabled/disabled by project admins.
  • Numerous bug fixes.
