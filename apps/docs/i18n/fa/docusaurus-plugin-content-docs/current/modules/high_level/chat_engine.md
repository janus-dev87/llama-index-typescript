# چت انجین (ChatEngine)

`undefined`

چت انجین یک راه سریع و ساده برای چت با داده ها در شاخص شماست.

```typescript
const retriever = index.asRetriever();
const chatEngine = new ContextChatEngine({ retriever });

// شروع چت
const response = await chatEngine.chat(query);
```
