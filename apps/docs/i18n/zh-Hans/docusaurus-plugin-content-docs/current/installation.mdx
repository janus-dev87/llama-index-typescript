# 安装和设置

确保您拥有 NodeJS v18 或更高版本。



## 使用 create-llama

开始使用 LlamaIndex 最简单的方法是使用 `create-llama`。这个命令行工具可以让您快速开始构建一个新的 LlamaIndex 应用程序，并为您设置好一切。

只需运行

<Tabs>
<TabItem value="1" label="npm" default>

```bash
npx create-llama@latest
```

</TabItem>
<TabItem value="2" label="Yarn">

```bash
yarn create llama
```

</TabItem>
<TabItem value="3" label="pnpm">

```bash
pnpm create llama@latest
```

</TabItem>
</Tabs>

即可开始。生成应用程序后，运行

```bash npm2yarn
npm run dev
```

启动开发服务器。然后，您可以访问 [http://localhost:3000](http://localhost:3000) 查看您的应用程序。

## 从 NPM 安装

```bash npm2yarn
npm install llamaindex
```



### 环境变量

我们的示例默认使用 OpenAI。您需要设置您的 OpenAI 密钥，如下所示：

```bash
export OPENAI_API_KEY="sk-......" # 请用 https://platform.openai.com/account/api-keys 上获取的密钥替换
```

如果您希望每次自动加载它，请将其添加到您的 .zshrc/.bashrc 中。

警告：不要将您的 OpenAI 密钥提交到版本控制中。
